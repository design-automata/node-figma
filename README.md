# node-figma

Zero-dependency wrapper around the Figma API for Node.js.

## Getting started
Grab dependencies:
```sh
npm i
```

Note: Export your Figma PAT in your [environment variables](src/config.ts) before proceeding.

Build:
```sh
npm run build
```

Run:
```sh
node dist
```
